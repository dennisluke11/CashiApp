Initial commit: Cashi Mobile App - Kotlin Multiplatform FinTech Payment App

Features:
- Send payment with validation
- Transaction history with real-time Firestore updates
- Multi-currency support (USD, EUR, GBP)
- Material Design 3 UI
- Extracted reusable components

Architecture:
- Kotlin Multiplatform (80% code sharing)
- MVVM architecture with clean layer separation
- Koin dependency injection
- Repository pattern
- Use cases for business logic
- expect/actual for platform-specific code

Backend:
- Node.js/Express REST API
- POST /payments endpoint
- POST /validate endpoint
- Firestore integration

Testing:
- BDD tests with Cucumber
- Unit tests with Kotest
- JMeter performance tests
- Appium UI tests
